# Sales Data Analytics Dashboard

## Project Overview
This project is a Sales Data Analytics Dashboard created using Microsoft Power BI. It provides insights into sales performance using interactive charts, KPIs, and filters.

## Tools Used
- Microsoft Power BI
- Microsoft Excel / CSV

## Dashboard Features
- Total Revenue
- Total Quantity
- Total Orders
- Average Order Value
- Revenue by Product
- Revenue by City
- Revenue by Category
- Revenue by Year
- Category Filter
- City Filter
